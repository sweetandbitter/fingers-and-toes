This is the Fingers and Toes assignment.

1. This website using HTML5, CSS3 and pure JavaScript.

2. User can input nonnegative integer as the end number, and invalid value will not pass the form validate check.

3. There are three buttons, start button will start the number count up and disable when count start. Restart button can restart count at anytime. Reset button will reset the input and counter.

4. Counter will increase every second.

5. Fingers and Toes will be highlight(in red), if counter can be both divisible by 3 and 5(I consider 0 fit this),  highlight fingers when the counter is divisible by 3, highlight toes when the counter is divisible by 5.

6. Using CSS3 media query to do responsive website.
